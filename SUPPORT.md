# Support Guide

Welcome to the Strix-Kali-AI support ecosystem! Before reaching out, please check the following resources to see if your query has already been resolved.

## 🔍 Self-Help Resources

* **Sandbox Fetch Issues**: If your agent freezes at startup, please see the [README Sandbox Optimization section](README.md#3-accelerated-sandbox-setup-domestic-mirror-fix) regarding the Docker local manual tagging fix.
* **Token Burning & Cost Management**: Check our Model Matrix inside the documentation folder to prevent heavy token drain when utilizing high-reasoning modes (`STRIX_REASONING_EFFORT="high"`).

## 💬 Communication Channels

### 1. GitHub Issues
* **Bug Reports**: If you encounter dependency crashes, CLI parsing failures, or system level bugs, please open an Issue using our standardized template.
* **Feature Requests**: Want to add support for a new LLM provider (e.g., Baichuan, Moonshot)? Open an Issue tagged with `feature-request`.

### 2. Discussions
For general architectural debates, sharing your own `--instruction` optimization recipes, or showcasing successful non-false-positive validation reports, head over to our **GitHub Discussions** tab.
